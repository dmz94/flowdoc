  • Getting Started

    Decant makes web articles easier to read by stripping away clutter and reformatting content with typography designed for readers with dyslexia. It is useful for anyone who prefers cleaner, distraction-free reading.

    Paste a URL or upload an HTML file. It works best with articles, blog posts, and long-form prose.

    Some websites load content in ways Decant cannot process. If a page appears blank or incomplete after conversion, try saving it from your browser and uploading it instead.

    Usage Notes

    • If a web page requires login, save it from your browser (File > Save As > HTML) and upload it instead.
    • The Share and View Original buttons are only available when converting from a URL.
    • If you uploaded a file, some links may not work because the original web address is not available.
    • The Share button copies a link to your clipboard. Send it to someone and they will see the same article, already converted.
    • Use the Reset button in Text controls to restore all settings to defaults.
    • To save as PDF, click Print and choose Save as PDF from your printer options. A dedicated PDF export is planned for a future release.
    • Tip: Some apps let you export content as HTML -- recipes from cooking apps, notes, saved articles. You can upload those files here too.

    About

    Decant is focused on making web content more accessible for readers with dyslexia. Support for other conditions, including color vision differences, is planned.

    Decant is open source under the MIT license.

    Developers: add Decant's conversion capabilities to your own app or workflow. See decant-cli on PyPI.
